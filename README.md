# 🚀 Claude Code Cost Estimator by JDiDev7

Una herramienta sencilla y eficaz para desarrolladores que utilizan la CLI de **Claude Code**. 
Calcula tus tokens y costes antes de ejecutar comandos para evitar sorpresas en tu factura de Anthropic.

## ✨ Características
- **Estimación Real:** Basada en líneas de código (LOC).
- **Modelos Actualizados:** Soporte para Claude 3.5 Sonnet y Haiku.
- **Privacidad:** Todo el cálculo se hace en el navegador (client-side).
- **Gratis:** Creado por y para la comunidad.

## 🛠️ Tecnologías
- HTML5 / Tailwind CSS
- JavaScript (Vanilla)
